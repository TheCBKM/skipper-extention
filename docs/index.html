Skip the ads. Keep the show.

Skipper is a Chrome extension that automatically skips or fast-forwards advertisements on Hotstar, Amazon Prime Video, and YouTube while you watch.

Add to Chrome Get support

Supported platforms

Hotstar

Automatically skips ad video segments when they appear in the player.

Prime Video

Skips and fast-forwards Amazon Prime Video ads with a simple on/off toggle.

YouTube

Skips platform ads and fast-forwards creator promo segments detected from captions.

How it works

  1. Install Skipper from the Chrome Web Store.
  2. Pin the extension to your toolbar.
  3. Open a supported streaming site and start watching.
  4. Use the Skipper popup to see which platform is active and adjust toggles anytime.

Skipper only runs on supported sites. It does not require an account and stores your preferences in Chrome sync storage.
